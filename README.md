# 🎓 Student Marks Predictor using Flask

## 📌 Project Overview

The Student Marks Predictor is a Machine Learning web application developed using Flask. It predicts a student's expected marks based on the number of study hours entered by the user. The application uses a pre-trained Machine Learning model to generate predictions and provides an easy-to-use web interface for users.

In addition, every prediction made by the application is stored in a CSV file for future analysis.

---

## 🚀 Features

- Predicts student marks based on study hours
- Simple and user-friendly Flask web application
- Uses a trained Machine Learning model
- Validates user input
- Stores prediction history in a CSV file
- Fast and lightweight application

## 📊 How It Works

1. User enters the number of study hours.
2. Flask receives the input.
3. The trained Machine Learning model predicts the marks.
4. The predicted marks are displayed on the webpage.
5. The prediction is saved to a CSV file.
